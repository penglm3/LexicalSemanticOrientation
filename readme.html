参考 Turney and Littman (2003) ，使用余弦相似度 (cosine similarity) 和正点交互熵 (positive pointwise mutual information, PPMI) 计算特定语域中全部词语的语义倾向 (semantic orientation)。

与 Turney and Littman (2003) 不同的是，余弦相似度使用的词向量由 word2vec 计算得来，而非 LSA (latent semantic analysis)。此外，使用了 PPMI，而非原文的PMI。

Turney and Littman (2003) 对语义倾向的抽象定义由下式给出：

SO_A(word)=∑pword∈PA(word,pword)−∑nword∈NA(word,nword)(1)SO\_A (word) = \sum_{pword\in P}A(word,pword)-\sum_{nword\in N}A(word,nword) \tag{1}SO_A(word)=pword∈P∑​A(word,pword)−nword∈N∑​A(word,nword)(1)

它衡量了特定词语 wordwordword 与给定一正向词集合 PPP 和负向词集合 NNN 之间关联性的差异。其中 pwordpwordpword 是正向词集合 PPP 中某词语，A(word,pword)A(word,pword)A(word,pword) 是 wordwordword 和 pwordpwordpword 之间的关联性函数。SO_A(word)SO\_A (word)SO_A(word) 是根据正向词集合 PPP 和负向词集合 NNN 判定的词语语义倾向，当 SO_A(word)SO\_A (word)SO_A(word) 取值相对较大且为正时，意味着词语 wordwordword 很可能具有正向含义，反之则可能有负向含义。

在 (1)(1)(1) 式基础上，以正点交互熵 PPMIPPMIPPMI 和余弦相似度 coscoscos 作为关联性函数 AAA，具体定义以下两种语义倾向函数：

SO_PPMI(word)=∑pword∈PPPMI(word,pword)−∑nword∈NPPMI(word,nword)(2)SO\_PPMI (word) = \sum_{pword\in P}PPMI(word,pword)-\sum_{nword\in N}PPMI(word,nword) \tag{2}SO_PPMI(word)=pword∈P∑​PPMI(word,pword)−nword∈N∑​PPMI(word,nword)(2)

SO_cos(word)=∑pword∈Pcos(word,pword)−∑nword∈Ncos(word,nword)(3)SO\_cos (word) = \sum_{pword\in P}cos(word,pword)-\sum_{nword\in N}cos(word,nword) \tag{3}SO_cos(word)=pword∈P∑​cos(word,pword)−nword∈N∑​cos(word,nword)(3)

wordiword_iwordi​ 和 wordjword_jwordj​ 的余弦相似度定义为词向量内积与模的比值:
